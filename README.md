| **Component** | **Model** |
| ------------- | --------- |
| CPU | AMD Ryzen 5 5600X @ 3.7GHz |
| Motherboard | ASUS Rog Strix B550F |
| RAM | 32GB (2 x 16GB) Asgard W2 @ 3200MHz @OC 3600MHz C16|
| Audio Chipset | ALC-S1220A |
| GPU | SAPPHIRE NITRO+ RX 580 8G G5 |

## What works
- Audio (`alcid=3`)
- Ethernet
- USB
- Wi-Fi
- Bluetooth
- iMessage, FaceTime

## Known issues
